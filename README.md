# nos.today
nos.today is a Nostr search web client. https://github.com/darashi/nos.today

Events are retrieved from the following relays:

- wss://search.nos.today (NIP-50 supported)
- wss://relay.nostr.band (NIP-50 supported)
- wss://relay.damus.io
- wss://nos.lol
